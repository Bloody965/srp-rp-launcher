# 🚀 Финальная настройка Railway с PostgreSQL

## ✅ Что уже сделано в коде:
- Добавлена поддержка PostgreSQL
- Код автоматически определяет какую БД использовать (PostgreSQL на Railway, SQLite локально)

## 📋 Шаги для настройки:

### 1. Добавь PostgreSQL в Railway

1. Открой Railway Dashboard → твой проект
2. Нажми **"+ New"**
3. Выбери **"Database"** → **"Add PostgreSQL"**
4. Railway автоматически создаст базу данных

### 2. Установи JWT секрет

1. Сгенерируй секретный ключ в PowerShell:
```powershell
-join ((65..90) + (97..122) + (48..57) | Get-Random -Count 64 | % {[char]$_})
```

2. Скопируй результат (64 символа)

3. В Railway перейди на вкладку **"Variables"** (твоего API сервиса)

4. Добавь переменную:
   - **Name:** `JWT__SECRETKEY`
   - **Value:** твой сгенерированный ключ

5. Нажми **"Add"** или **"Save"**

### 3. Задеплой обновленный код

```bash
cd D:\лаунчер
git add .
git commit -m "Add PostgreSQL support and fix session persistence"
git push
```

Railway автоматически задеплоит новую версию.

### 4. Проверь что всё работает

После деплоя проверь логи в Railway:
- Должно быть сообщение: `Using PostgreSQL database`
- Не должно быть ошибок подключения к БД

### 5. Протестируй

1. **Зарегистрируйся** в лаунчере
2. **Закрой** лаунчер
3. **Открой** снова → должен остаться залогинен ✅
4. **Задеплой** еще раз (любое изменение) → всё еще залогинен ✅

## 🎉 Готово!

Теперь:
- ✅ PostgreSQL база данных (не удаляется при деплое)
- ✅ Постоянный JWT секрет (токены работают после деплоя)
- ✅ Пользователи НЕ вылетают из аккаунта
- ✅ Все данные сохраняются

## 🔍 Проверка переменных в Railway

Убедись что установлены:

```
DATABASE_URL=postgres://... (автоматически от PostgreSQL)
JWT__SECRETKEY=твой_64_символьный_ключ (вручную)
```

## ⚠️ Важно

После первого деплоя с PostgreSQL база будет пустая. Все пользователи должны будут зарегистрироваться заново **один раз**. После этого данные будут сохраняться навсегда.

## 🔄 Миграция данных (опционально)

Если хочешь перенести существующих пользователей из SQLite в PostgreSQL:

1. Экспортируй данные из старой SQLite базы
2. Импортируй в новую PostgreSQL базу через Railway Query

Но проще просто начать с чистой базы.
