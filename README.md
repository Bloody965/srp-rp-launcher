## 📖 Описание

**SRP-RP Launcher** - защищенный кастомный лаунчер для Minecraft 1.20.1 с Forge, выполненный в стиле SRP-RP проекта.

### ⚡ Особенности

- 🎨 **Современный дизайн** - в стиле сайта SRP-RP с градиентами и эффектами
- 🔐 **Система безопасности** - регистрация, JWT токены, защита от взлома
- ⚙️ **Forge 47.3.0** - поддержка модов для версии 1.20.1
- 🚀 **Автоматическая установка** - скачивание и настройка всех необходимых файлов
- 📦 **Защищенные модпаки** - SHA256 проверка целостности файлов
- 📊 **Адаптивный интерфейс** - можно изменять размер окна
- 🖥️ **Полноэкранный режим** - для игры
- 💾 **Без консоли** - чистый запуск EXE файла
- 👤 **Профиль пользователя** - отображение имени и email
- ❌ **Визуальные ошибки** - понятные сообщения об ошибках в интерфейсе

## 🔐 Система безопасности

### Защита включает:
- ✅ **Регистрация и авторизация** с проверкой сложности пароля
- ✅ **Bcrypt хеширование** паролей (невозможно расшифровать)
- ✅ **JWT токены** с истечением срока (24 часа)
- ✅ **Rate limiting** - защита от брутфорса (5 попыток в 15 минут)
- ✅ **Защита от SQL инъекций** - валидация всех входных данных
- ✅ **SHA256 проверка** файлов модпаков
- ✅ **Логирование** всех действий с IP адресами
- ✅ **Ban система** для блокировки пользователей
- ✅ **Whitelist система** (опционально)
- ✅ **Сброс пароля** через email

## 🚀 Быстрый старт (локальная разработка)

### 1. Запусти API сервер (обязательно!)

Открой PowerShell/CMD:
```bash
cd D:\лаунчер\src\ApocalypseLauncher.API
dotnet run
```

API будет работать на **http://localhost:5000**

### 2. Запусти лаунчер

Запусти: `D:\лаунчер\publish\ApocalypseLauncher.exe`

## 📖 Использование

### Первый запуск

1. **Запусти API сервер** (см. выше)
2. **Запусти лаунчер**
3. **Нажми "Нет аккаунта? Регистрация"**
4. **Заполни данные:**
   - Имя игрока (3-16 символов, только буквы/цифры/_)
   - Email
   - Пароль (минимум 8 символов, заглавная, строчная, цифра)
5. **Нажми "РЕГИСТРАЦИЯ"**
6. **После успешной регистрации** ты автоматически войдешь
7. **Нажми "УСТАНОВИТЬ"** для загрузки Minecraft
8. **Нажми "ИГРАТЬ"** после установки

### Следующие запуски

1. Запусти API сервер
2. Запусти лаунчер
3. Введи имя и пароль
4. Нажми "ВОЙТИ"

## 📦 Добавление модпака

2. Получи SHA256 хеш файла

4. Положи файл в папку `modpacks/`

## 🔍 Тестирование защиты

См. полную инструкцию в **ТЕСТИРОВАНИЕ_БЕЗОПАСНОСТИ.md**

Быстрый тест:
- Попробуй зарегистрироваться с паролем "123" - должна быть ошибка
- Попробуй войти с неверным паролем 6 раз - блокировка после 5 попыток
- Проверь логи в базе данных `AuditLogs`

## 🐛 Известные проблемы

1. **API не запускается** - проверь что порт 5000 свободен
2. **Лаунчер не подключается** - убедись что API запущен
3. **Java не найдена** - установите Java 17+ и настройте JAVA_HOME
4. **Токен недействителен** - войди заново (токен истекает через 24 часа)

## 🌐 Production (публичный сервер)

Для деплоя на публичный сервер:
1. Залей API на Railway.app или Render.com
2. Получи публичный URL
3. Измени URL в лаунчере с `http://localhost:5000` на твой домен
4. Настрой SSL сертификат
